# steganography-llm-watermarking
Concealing secrets in plain sight using Llama 2 and conditional text watermarking.

This project explores advanced steganography techniques applied to Large Language Models (LLMs). We implemented and analyzed conditional watermarking methods using Llama 2 to hide confidential data within AI-generated news articles.

Key Features:

- Controlled Generation: Secret encoding by manipulating token probability distributions (Green/Red lists).

- Detection & Recovery: Algorithms designed to identify modified text and extract hidden messages without affecting semantic coherence.

- Robustness Analysis: Evaluation of resistance against common text-editing attacks and human detection.
