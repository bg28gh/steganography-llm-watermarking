# Steganography with Language Models in News Articles 🕵️‍♂️📄

This project, developed during my time at **Technische Universität Berlin**, explores the intersection of **Large Language Models (LLMs)** and **Steganography**. The goal is to discreetly hide and recover secrets within human-readable text generated by AI.

## 🚀 Project Overview
As LLMs become ubiquitous, the ability to distinguish between human and machine-generated text—and to use that text as a secure communication channel—is becoming critical. This research implements **conditional watermarking** techniques using the **Llama 2** model.

## 🧠 Key Research Areas
* **Conditional Watermarking:** Manipulating token probability distributions (Green/Red lists) during text generation to encode hidden information.
* **Secret Recovery:** Algorithms designed for algorithm-aware "co-conspirators" to extract hidden messages with high accuracy.
* **News Article Contextualization:** Using news articles as a dependable source to ensure the steganographic text remains semantically coherent and indistinguishable from natural human writing.
* **Robustness Testing:** Evaluating how editing or human detection affects the integrity of the hidden data.

## 🛠️ Tech Stack
* **Models:** Llama 2 (Meta AI).
* **Environment:** Python-based research environment.
* **Concepts:** Natural Language Processing (NLP), Information Theory, Cybersecurity.
